# RTL Hebrew — כיוון כתיבה מימין לשמאל

## פקודה: `/rtl-hebrew`

סקיל זה מפעיל מצב כתיבה עברית מלאה — כל תגובה כתובה מימין לשמאל עם Unicode RTL markers.

---

## הוראות לClaude — חובה ליישם

כשסקיל זה פעיל:

1. **כל טקסט עברי** מתחיל עם RTL mark (`‏` — U+200F) כדי לאלץ כיוון נכון
2. **מבנה תגובות** — כותרות, רשימות, טבלאות — הכל בנוי RTL
3. **קוד + הסברים** — ההסבר בעברית מופיע מימין, הקוד בלוק נשאר LTR (טבעי)
4. **אין להתחיל שורה בתו LTR** כשהתוכן עברי

---

## תבנית תגובה RTL

### כותרות:
‏## כותרת בעברית — תמיד ב-RTL

### רשימות:
‏- פריט ראשון
‏- פריט שני
‏- פריט שלישי

### טבלאות — כותרות בעברית ראשונות:
| תיאור | ערך |
|-------|-----|
| ‏נתון ראשון | 123 |

### הסבר + קוד:
‏הקוד הבא מבצע את הפעולה:
```javascript
// קוד — נשאר LTR תמיד
const x = 1;
```

---

## Unicode RTL Characters

| תו | Unicode | שימוש |
|----|---------|-------|
| `‏` | U+200F | RTL Mark — מוסיפים לפני טקסט עברי |
| `‎` | U+200E | LTR Mark — לאחר קוד inline בתוך עברית |
| `‭` | U+202A | LTR Embed — לבלוקים אנגלים בתוך עברית |

---

## דוגמה — תגובה נכונה vs שגויה

**שגוי (LTR):**
```
The booking system works like this:
1. User selects service
2. User picks date
```

**נכון (RTL):**
```
‏מערכת ההזמנות עובדת כך:
‏1. משתמש בוחר שירות
‏2. משתמש בוחר תאריך
```
